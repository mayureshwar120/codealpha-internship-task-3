
# Bhagat Singh Typography Poster

## Project Overview

This project is a typography-based poster dedicated to Bhagat Singh, one of India's most influential freedom fighters. The design uses typography as the primary visual element to create a powerful and impactful representation of his legacy, courage, and patriotism.

## Design Concept

The poster combines meaningful words, quotes, and typographic elements to visually portray the spirit of Bhagat Singh. The design focuses on creating a strong emotional connection while maintaining visual balance, hierarchy, and readability.

## Objectives

* Explore creative typography techniques.
* Communicate a historical figure's personality through text-based design.
* Apply principles of visual hierarchy, contrast, and composition.
* Create an engaging and memorable poster design.

## Tools Used

* Figma / Adobe Photoshop / Canva (update according to the tool used)
* Typography Design Principles
* Graphic Design Fundamentals

## Key Design Elements

* Bold typography to represent strength and courage.
* Strategic use of contrast and spacing.
* Balanced composition for visual appeal.
* Historical and patriotic theme.

## Outcome

The final poster successfully demonstrates the use of typography as a powerful design medium while honoring the legacy of Bhagat Singh. The project showcases creativity, visual storytelling, and fundamental graphic design skills.
